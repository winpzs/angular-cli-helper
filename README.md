# angular-cli-helper

把angular-cli结合到VS code的辅助工具.

## 运行环境

1. nodejs
2. npm install -g @angular/cli

## 相关配置

1. angularclihelper.generateConfig: 生成指令配置
2. angularclihelper.buildParam: ng build参数配置
3. angularclihelper.serveParam: ng serve参数配置
